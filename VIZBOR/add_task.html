Тег dl (description list) создает список с подробным объяснением его элементов.
Работает в связке с тегами dt и dd.
Тег dt (description term) используется при создании списка dl и создает элемент с названием термина, который будет объяснен в теге dd.
Тег dd (description details) используется при создании списка dlи создает элемент с описанием термина, который будет указан в теге dt.
